# Ventoo - E-commerce

 - Aplicación móvil que simula un e-commerce, desarrollada como proyecto full stack para practicar la integración entre frontend, backend y base de datos.

 - <a href="https://github.com/SaraCuellar89/APK_Ventoo" target="_blank"><strong>¡Descarga el apk!</strong></a> 
---

## Tecnologías Utilizadas
- **React Native**
- **TypeScript**

## Ejecutarlo de forma local
 - Descargar el Proyecto.  
 ```
 git clone https://github.com/SaraCuellar89/Aplicacion_Movil_Ventoo.git
 ```
   - Abrir una terminal.
   ```javascript
   //Intalar dependecias:
   npm install
   ```
   ```javascript
   //Ejecutar Metro:
   npm start
   ```
   - Abrir otra terminal.
   ```javascript
   //Ejecutar la aplicacion:
   npm run android
   ```
